The Json File of "Specification" Directory can be used as the body 
of the add-openroadm-operationnal-modes-to-catalog RPC to fill
in the service Data store of the ROADM network controller , the 
operational-mode-catalog. After this operation, it will then contain
the translation of the latest version of OpenROADM optical specifications
(available at the time yang models are publically realeased).
